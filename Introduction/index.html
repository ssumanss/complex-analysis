    Definition and Algebra

    In this section we will define complex numbers as an extension of as the following 

    from sympy import *
init_printing()

    a,b,c,d = symbols("a b c d")
x,y = symbols("x y")

    Algebra of Complex Number¶

    Addition of two complex numbers are done as follows

    ad = (a+b*x)+(c+d*x)
Eq(ad, collect(ad,x))

    Multiplication is done as follows

    mul = (a+b*x)*(c+d*x)
Eq(mul, expand(mul))


    irr = x**2+1
quo, rem = pdiv(mul,irr)
collect(rem,x)
